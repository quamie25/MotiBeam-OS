üöÄ HOW TO START MOTIBEAM

Option 1: Auto-Start (Power On)
1. Plug in Raspberry Pi power
2. Wait ~1 minute
3. MotiBeam boot screen appears on projector automatically
4. Main menu shows up after boot

Option 2: Manual Start (via SSH)

From your Mac Terminal:

# Connect to Pi
ssh motibeam@motibeamos.local

# Navigate to project
cd ~/MotiBeam-OS

# Activate Python environment
source motibeam-env/bin/activate

# Run MotiBeam
python3 motibeam_app.py
